\CWHeader{Лабораторная работа \textnumero 3}

\CWProblem{
Необходимо провести исследование скорости выполнения и потребления
оперативной памяти. В случае выявления ошибок или явных недочётов, требуется
их исправить.

В данной работе для оценки потребления оперативной памяти будет использована утилита valgrind, а для оценки скорости работы программы - утилита gprof.
}
